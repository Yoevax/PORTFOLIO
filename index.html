PORTFOLIO

FIND ME

Find me is my first dynamic project made during my training at interface3. The website was made with PHP, MySql for the backend and JavaScript, HTML and CSS for the frontend. The idea of Find me is to help the user to find a movie by answering 3 questions. Depending on the answers chosen, a selection of films will be proposed.

MATCHING CODE GAME

Matching code game is a card game (memory style) that was made in a group in two days during the hackathon 2022 which had for theme "coding and decoding". Matching game is a site that allows you to learn the basics of programming with PHP and HTML markup language and CSS. There are 3 rounds per language, from level 1 to level 3, the principle is simple: match the cards in a given time!

AGREGATOR

Application that's shows time, date, weather, float rates, trains departures from Brussels north and the Nasa's picture of the day. Thanks to NodeJs, I was able to compile all the codes to have an application that fetches information (iRail, Nasa, float rates, weather, rss feeds ...) on the Internet to save them in a server-side cache in JSON, so as to be able to send them to a client formatted from an EJS template, CSS and integrating images.
